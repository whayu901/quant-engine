# Glossary (Kadence / market research)

- **KIA** — Key Information Areas; the objectives a questionnaire must cover.
- **CAPI / CAWI** — Computer-Assisted Personal (tablet, in-person) / Web Interviewing (online).
- **SurveyToGo (Dooblo)** — Kadence's primary CAPI fieldwork + QC platform (GPS, silent-audio,
  speeder/straightlining flags).
- **tSurvey** — Telkomsel-powered online/CAWI panel Kadence uses.
- **KOINS** — *Kadence Online Insight System*; Kadence's own proprietary online survey platform.
- **QPSMR** — tabulation / data-processing software used by Kadence DP.
- **Code frame** — the coding scheme mapping open-ended answers / variables for tabulation.
- **Quota / cell** — sample targets per group; field closes a cell when its target is met.
- **QDT** — Questionnaire Development Team (the survey programmers / scripters).
- **DP** — Data Processing (cleaning, tabulation, delivery QC).
- **RE** — Research Executive.
- **Curbstoning** — fabricating interview data without actually interviewing; what Fieldwork QC catches.
- **BHT / U&A** — Brand Health Tracking / Usage & Attitude studies.
