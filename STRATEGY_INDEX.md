## KEY DECISIONS MADE (Don't Re-open These)

1. **MVP Scope**: Auth, dashboard, upload, transcript viewer, analysis grid, evidence panel, chat, export, mobile
   - Not included: Video player, clips, knowledge base, live recording, concept testing
   - These are Phase 1A, not MVP

2. **Tech Stack**: Next.js, React 18, TypeScript, Tailwind, FastAPI (backend)
   - Locked for 12 months; don't change

3. **Pricing**: 3 tiers (Starter SGD 49, Pro SGD 99, Ent SGD 249+)
   - Locked for 6 months; don't adjust without board discussion

4. **Target Market**: Southeast Asia first (SG, ID, MY, TH, VN, PH)
   - Not US, not global, not European
   - Build FOR SEA, not adapted for SEA

5. **Go-to-market**: Singapore first (Week 1), Indonesia (Week 2), others sequential
   - Don't launch all countries simultaneously (resource constraint)

6. **Competitive posture**: Local-first, not feature-matching
   - Don't try to match Coloop feature-for-feature
   - Win on code-mixing, data residency, pricing, local ecosystem
